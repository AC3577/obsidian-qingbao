## 二、推荐选题及角度

### 选题1：【小红书】AI工具爆炸时代的「数字疲惫」，你中招了吗？

**情绪切入口**：AI疲惫感  
**爆款因素**：反直觉——不写AI多厉害，而是写「AI太多让人烦」  
**目标受众**：25-35岁，被AI工具轰炸的职场人和创作者  
**内容形式**：共鸣吐槽帖 + 数据支撑 + 缓解建议  

**钩子句**：「救命，AI多到让人焦虑，我实在受够了」——这句话说出了多少人的心声？  

**内容结构**：
1. 现象描述：你是否也有「AI疲劳症」？
2. 数据支撑：B站视频引爆的「AI疲惫」讨论（附截图）
3. 原因拆解：工具太多、学习成本太高、获得感太低
4. 行动建议：与其追工具，不如追结果（减法策略）

**爆款概率**：⭐⭐⭐⭐⭐（情绪精准 + 数据新鲜 + 9小时内新热点）

---

### 选题2：【公众号深度】AI真的在抢你工作吗？2026新职业地图发布

**情绪切入口**：职业焦虑 + 出路感  
**目标受众**：25-40岁职场人、创业者、求职者  
**内容形式**：深度长文（3000字+）  
**数据锚点**：人社部新职业名录 + 麦肯锡报告 + 机器人人才需供比  

**钩子句**：「机器人领域人才供需比5.2:1，意味着什么？」  

**内容结构**：
1. 现象：AI裁员潮是真的吗？（数据打脸：仅1%裁员真正因AI）
2. 真相：不是岗位消失，是「工作解剖结构」重组
3. 新机会：人社部新职业名录解读（生成式AI动画制作员、提示词工程师等）
4. 人机协同：未来3年最有价值的技能是什么？
5. 行动清单：普通人如何准备

**深度触发**：内容深度足够，建议同步投递 media-mp 触发深度研究流程

### 选题3：【小红书】为什么「AI取代人类」是个伪命题——2026年真正危险的反而是它

**情绪切入口**：反主流叙事 + 好奇心  
**目标受众**：对AI话题有关注但被焦虑淹没的用户  
**内容形式**：观点输出型笔记  
**差异化**：市场上99%写AI取代，这篇写「取代论是错的，真正危险的是别的东西」  

**钩子句**：「所有人都说AI会取代你，但没人告诉你：真正危险的，恰恰是你相信了这句话」  

**内容结构**：
1. 热点引入：OpenAI前研究员的末日时间表
2. 反转：裁员数据告诉你真相（仅1%是AI导致）
3. 真正危险：Gartner揭示的AI九大坑
4. 结论：焦虑本身才是最大的敌人

### 选题4：【小红书】算了，不追AI了：一位内容创作者的「AI减法」实验

**情绪切入口**：AI疲惫 + 成长叙事  
**目标受众**：被AI工具焦虑裹挟的创作者  
**内容形式**：个人实验/故事型  

**钩子句**：「扔掉了5个AI工具后，我的创作效率反而翻倍了」  

**内容结构**：
1. 背景：我曾经追了哪些AI工具（清单）
2. 问题：工具越多，越焦虑
3. 实验：30天「AI减法」挑战
4. 结果：扔了什么，留了什么，效果如何
5. 结论：少即是多

**爆款潜力**：故事型 + 实验型，在小红书一直表现稳定
