## 钩子公式库

### 1. 诊断命名式
**为何有效**：把模糊痛苦贴上权威标签，产生"终于有词描述我的状态"的释放感，评论/共鸣率最高。

**模板**：
```
你可能得了「___」——不是病，是没有人告诉你的名字
「___」不是矫情，是你的神经系统在求救
[年龄]才知道，这叫「___」，不是我的错
```

---

### 2. 反常识颠覆式
**为何有效**：与主流"努力/正能量论"对着干，制造认知冲突，迫使读者停下来看完。两个赛道都高频出现。

**模板**：
```
不是你不努力，是你搞错了方向——「___」才是真正有用的
一直逼自己「___」，只会更___
[主流做法]没用，真正管用的是这一步
```

---

### 3. 数字承诺式
**为何有效**：大脑对具体数字产生"可完成感"，激发收藏动作，两个赛道一致验证。

**模板**：
```
[N]条能帮你「___」的方法（存下来用）
[N]分钟，把「___」变成「___」
[年龄]后我只保留这[N]件事
```
> 数字选3/4/5/8/12/15/101，不要用6/7/9。

---

### 4. 年龄锚定+身份落差式
**为何有效**：年龄触发阶段性焦虑，身份落差制造好奇，双重钩叠加点击欲。

**模板**：
```
[年龄]发现的道理，[更大年龄]才后悔没早学
同样是[年龄]：她「___」，我「___」——差在哪？
[年龄]太焦虑？我跟「[反差身份]」学到了什么
```

---

### 5. 情绪验证式
**为何有效**：用"我最近有种感觉……"式的模糊叙述激活"我也是！"共鸣，评论参与度极高，适合开启互动话题。

**模板**：
```
最近突然有种感觉：「___」，你有吗？
不知道为什么，开始「___」了，感觉对了
有没有人和我一样，突然不想再「___」了
```

## 叙事结构模板

### 结构一：工具型（收藏驱动）
**适用场景**：方法论、清单、可操作步骤类内容

```
[痛点场景/症状描述，让读者对号入座]
     ↓
[一句话解释"原来是这个原因"——科学/玄学任选]
     ↓
[N条具体方法，每条一行，加粗关键词]
     ↓
[行动金句收尾 + "存下来试试"]
```

> 节奏要求：每屏只放一个要点，快节奏翻页。

---

### 结构二：故事型（评论驱动）
**适用场景**：个人经历、认知转变、反叙事类内容

```
[我经历过的具体低谷时刻，细节要真实]
     ↓
[转折点：我发现/学到/意识到了「___」]
     ↓
[中段埋一句："你是不是也……" → 主动邀请留言]
     ↓
[认知升华成一句可被转发的金句]
     ↓
[开放式问题结尾，不给答案，让读者接话]
```

> 节奏要求：情绪要有低谷再拉升，不能全程平稳。

## 情绪设计原则

**原则一：开头必须命名痛苦，不能直接给方法**
读者先要感到"被看见"，才会接受你的方法。跳过共鸣直接讲干货=没人听。

**原则二：中段必须给"不是你的错"的解释**
把自责转向外因（神经系统/环境/认知模式），情绪从羞耻→接纳，这是用户愿意分享的节点。

**原则三：结尾必须给掌控感，而不是开放式希望**
结尾要有一个可以立刻做的动作或一句可以立刻记住的话。"会好的"不够用，"你现在可以做这一件事"才有收藏价值。

## 我的禁区

1. **纯鸡汤无内核**：只有"你很好""相信自己"，没有机制解释或可操作步骤——读者会觉得空洞，不收藏不评论。

2. **中间地带人设**：既不够个人（没有真实经历），又不够专业（没有知识框架）。素材明确指出这是互动最差的区间。

3. **跳过低谷直接赋能**：开篇直接正能量，没有"被看见"的共情时刻——读者感到被说教，不是被理解。

4. **玄学不落地**：讲显化/宇宙但没有连接到具体行动或心理机制——在目标受众中可信度会快速流失。

5. **恐吓式流量**：用"你有这些症状说明病了"制造恐惧但不给出路——可以引发焦虑共鸣，但会让人关掉页面而不是留下。
